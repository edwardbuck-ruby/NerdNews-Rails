== سایت خبری

هدف از ایجاد این پروژه، ساخت یک نرم‌افزار وب برای گذاشتن خبرهای مختلف از دنیای آی‌تی و فناوری است. این پروژه به‌طور کامل آزاد و متن‌باز است که توسط چارچوب نرم‌افزاری وب روبی آن ریلز نوشته می‌شود.

این پروژه تجت اجازه‌نامهٔ GPL نسخهٔ ۳ منتشر می‌شود.
